–°–∞–º—ã–π –ø—Ä–æ—Å—Ç–æ–π —Ü–∏–∫–ª ‚Äî –º–æ–∂–Ω–æ –æ–±—ä—è–≤–∏—Ç—å —Ç–æ–ª—å–∫–æ —É—Å–ª–æ–≤–∏–µ, –∞ –æ—Å—Ç–∞–ª—å–Ω–æ–µ —Å–¥–µ–ª–∞—Ç—å –≤–Ω—É—Ç—Ä–∏ —Ü–∏–∫–ª–∞:

// –∏–∑–º–µ–Ω—è–µ–º–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –¥–ª—è —Ü–∏–∫–ª–∞
mut count := 10
// –ø–æ–∫–∞ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –±–æ–ª—å—à–µ 0 ‚Äî —Ü–∏–∫–ª —Ä–∞–±–æ—Ç–∞–µ—Ç
for count > 0 {
// –≤—ã–≤–æ–¥–∏–º —Ç–µ–∫—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π
println(count)
// —É–º–µ–Ω—å—à–∞–µ–º –µ—ë –Ω–∞ –µ–¥–∏–Ω–∏—Ü—É
count = count - 1
}

–§—É–Ω–∫—Ü–∏–∏#

–ü—Ä–∏ –æ–±—ä—è–≤–ª–µ–Ω–∏–∏ —Ñ—É–Ω–∫—Ü–∏–∏ –Ω—É–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å:

  • –∫–∞–∫–æ–≥–æ —Ç–∏–ø–∞ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è –≤ —Ñ—É–Ω–∫—Ü–∏—é –∫–∞–∫ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã;
  • –∫–∞–∫–æ–≥–æ —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ñ—É–Ω–∫—Ü–∏—è.
// —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –ø–µ—Ä–µ–º–Ω–æ–∂–∞–µ—Ç –¥–≤–∞ —á–∏—Å–ª–∞
// —É –Ω–µ—ë –¥–≤–∞ –≤—Ö–æ–¥–Ω—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ ‚Äî x –∏ y, –¥—Ä–æ–±–Ω—ã–µ —á–∏—Å–ª–∞
// —Ñ—É–Ω–∫—Ü–∏—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ç–æ–∂–µ –¥—Ä–æ–±–Ω–æ–µ —á–∏—Å–ª–æ
fn multiply(x,y f64) f64 { // float - f64
res := x * y
// –≤–æ–∑–≤—Ä–∞—â–∞–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ä–∞–±–æ—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏
return res
}
// –Ω–∞—á–∞–ª–∞—Å—å –æ—Å–Ω–æ–≤–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è
fn main() {
// –º–Ω–æ–∂–∏—Ç–µ–ª–∏
x := 15.0
y := 120.6
// –ø–æ–ª—É—á–∞–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ä–∞–±–æ—Ç—ã —Ñ—É–Ω–∫—Ü–∏–∏ –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é
mult := multiply(x,y)
// –≤—ã–≤–æ–¥–∏–º –æ—Ç–≤–µ—Ç
println(mult)
}

–°—Ç—Ä—É–∫—Ç—É—Ä—ã#

–í V –Ω–µ—Ç –ø—Ä–∏–≤—ã—á–Ω—ã—Ö –¥–ª—è –û–û–ü –∫–ª–∞—Å—Å–æ–≤, –Ω–æ –µ—Å—Ç—å —Å—Ç—Ä—É–∫—Ç—É—Ä—ã. –¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–ª—è –æ–±—ä—è–≤–ª–µ–Ω–∏—è –∫–ª–∞—Å—Å–∞ –∏ —á–µ—Ä–µ–∑ –Ω–∏—Ö —Å–æ–∑–¥–∞–≤–∞—Ç—å –Ω–æ–≤—ã–µ –æ–±—ä–µ–∫—Ç—ã, –Ω–æ –Ω—É–∂–Ω–æ —Å–ª–µ–¥–∏—Ç—å –∑–∞ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–æ–º.

–°—Ç—Ä—É–∫—Ç—É—Ä–∞ ‚Äî —ç—Ç–æ –∫–æ–≥–¥–∞ –º—ã –æ–±—ä–µ–¥–∏–Ω—è–µ–º –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑–Ω–æ—Ä–æ–¥–Ω—ã—Ö (–∏–ª–∏ –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö) —Ç–∏–ø–æ–≤ –¥–∞–Ω–Ω—ã—Ö –≤ –æ–¥–Ω–æ —Ü–µ–ª–æ–µ –∏ —Ä–∞–±–æ—Ç–∞–µ–º —Å –Ω–∏–º –∫–∞–∫ —Å –µ–¥–∏–Ω—ã–º –æ–±—ä–µ–∫—Ç–æ–º. –ü–æ—á—Ç–∏ –∫–∞–∫ –∫–ª–∞—Å—Å—ã –∏ –æ–±—ä–µ–∫—Ç—ã, —Ç–æ–ª—å–∫–æ –±–µ–∑ –∫–ª–∞—Å—Å–æ–≤ –∏ –æ–±—ä–µ–∫—Ç–æ–≤.

struct Point {

}
